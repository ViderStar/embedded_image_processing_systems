# Лабораторная работа 2 — Неразделимое 2-D преобразование

## Теория

В Lab 1 реализовано **1-D** преобразование CDF 9/7 с фиксированной запятой.  
Для обработки изображений необходимо **2-D** преобразование.

### Разделимый (separable) подход

Классический 2-D DWT выполняется как последовательность двух 1-D преобразований:

```
Y = Θ · X · Θᵀ
```

1. Преобразование по строкам
2. Преобразование по столбцам

Результат — четыре подполосы: **LL**, **LH**, **HL**, **HH**.

### Неразделимый (non-separable) подход

Блок изображения `X` размером `n×n` можно представить как вектор `x` длины `n²`.  
2-D преобразование записывается как одно матричное умножение:

```
y = Θ̈ · x
```

где матрица неразделимого преобразования:

```
Θ̈ = D(Θ) · P · D(Θ) · P
```

- **D(Θ)** — блочно-диагональная матрица с `n` копиями 1-D матрицы `Θ` на диагонали
- **P** — матрица перестановки, реализующая транспонирование блока (`tv(Xᵀ) = P · tv(X)`)

Матрица `Θ̈` **эквивалентна** разделимому преобразованию `Θ ⊗ Θ` при правильном упорядочивании коэффициентов, но вычисляется как **единая операция** без явного разделения на строки и столбцы.

### Отличие от разделимого метода

| Свойство | Разделимый | Неразделимый |
|----------|-----------|--------------|
| Вычисление | Строки → столбцы | Одна матрица `n²×n²` |
| Память | Промежуточный буфер `n×n` | Можно обрабатывать блоками |
| Гибкость | Ограничена | Позволяет обобщить на неразделимые фильтробанки |
